# NativeBase
React Native and Firebase todo app made for personal use
## Installation

Jump on a terminal
```bash
git clone https://github.com/neilbateman/NativeBase.git
cd nativebase
expo start
```
## Usage

Should work on webview, otherwise get the Expo app and scan the QR code served by expo
